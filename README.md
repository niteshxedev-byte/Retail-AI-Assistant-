# Retail AI Assistant

This repository contains both the frontend and backend for the Retail AI Assistant.

## Project Structure

- **[frontend/](frontend/)**: A React/TypeScript application with Tailwind CSS for the user interface.
- **[backend/](backend/)**: A Node.js application with Socket.IO and LangChain for the AI logic.
